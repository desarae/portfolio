Welcome to Portfolio Template for Blogspot

This is a set of themes and templates created for Blogger Blogspot.

Designer Templates

The template folder includes a set of blogger style templates. The templateview folder is the HTML version, most versions do not have an HTML version saved.

To implement the template:

  • Go to your blogger site
  • Click on the template tab in the admin panel
  • Select Edit HTML
  • Copy the code and paste the desired code from the template

Creating pages manually

Visit the Blogger admin panel. Pages will need links within the template, but posts will automatically be added. For theme bg-3 it is recommended that only 1 post per date is added. Otherwise the date wrapper will offset on screens over 1181px.
